# P-06: 実機検証とログ取得

- **渡す先**: Claude Code
- **タイミング**: APK インストール後
- **目的**: 実機で動作確認し、問題があればログを取得

---

## 1. 基本動作確認チェックリスト

以下を順番に実行し、結果を記録してください:

- [ ] アプリが起動する（クラッシュしない）
- [ ] ホーム画面が表示される
- [ ] 各タブ/画面に遷移できる
- [ ] コア機能（{アプリの主要機能}）が動作する
- [ ] データが保存される（アプリ再起動後も残る）
- [ ] 戻るボタンで期待通りの画面に戻る
- [ ] ダークモードで表示が崩れない
- [ ] 文字が切れたり重なったりしていない

## 2. ログ取得コマンド

```bash
# リアルタイムログ（アプリのみ）
adb logcat --pid=$(adb shell pidof {PACKAGE_NAME}) \
  | tee logcat_$(date +%Y%m%d_%H%M%S).log

# クラッシュログ
adb logcat -b crash -d > crash_$(date +%Y%m%d_%H%M%S).log

# メモリ使用量
adb shell dumpsys meminfo {PACKAGE_NAME} \
  > meminfo_$(date +%Y%m%d_%H%M%S).txt

# スクリーンショット
adb exec-out screencap -p > screenshot_$(date +%Y%m%d_%H%M%S).png
```

## 3. 問題発見時

問題を発見したら **P-08（実機バグトリアージ）** に移行する。

P-08 の Phase 1（Claude Code による解析）を開始し、
根本原因を特定してから自分で修正を実装すること（Claude Code 単独運用）。
