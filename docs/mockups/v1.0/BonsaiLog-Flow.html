プレーン日本語版（小中学生にも読める）
①アプリを開くと Splash ➜ Welcome。「はじめる」を押すと 言語 ➜ ATT（広告追跡について）➜ UMP（EU/UK/CH のみ）➜ 通知許可 へ。スキップで最短 0 タップで Home。
②初めての人は Home Empty（空っぽ画面）に着地。大きな「盆栽を登録」ボタンで 新規登録モーダル。
③すでに盆栽がある人は Home リスト。盆栽カードをタップすると 盆栽詳細（初期は 履歴タブ）へ移動。
④下のタブバーは 「盆栽 / 予定 / ふりかえり / 設定」 の 4 本（v1.7 で「記録」→「ふりかえり」）。予定 はカレンダー、ふりかえり は CareHub（水やり履歴・針金がけ一覧・盆栽検索 の 3 カード）、設定 はアカウント・出力・課金・バックアップ。
⑤Home ヘッダの 「複数選択」 ボタンで 複数選択モード（カード長押し でも可）。下部のツールバーから タグ追加 / 一括記録 / 予定追加。v1.10 で同じヘッダの 検索アイコン ➜ 盆栽検索画面に直接ジャンプ、設定アイコン ➜ 設定画面に直接ジャンプ（D9/D10 解消、装飾だけのナビ UI を撲滅）。
⑥盆栽詳細では 履歴 / 予定 / 基本情報 の 3 タブ。編集 は基本情報タブで完結（v1.1）、または 「・・・」➜ 基本情報を編集 ショートカット（v1.5）。「・・・」メニュー は 基本情報を編集 / PDF / アーカイブ の 3 項目（共有は v1.5 で削除）。
⑦記録の入口（v1.5）：詳細画面 右下 「＋作業を記録」FAB を常設。タップ ➜ 作業種別 Sheet（14 作業）➜ 作業別フォーム。履歴を見ながらその場で記録できる。
⑧出力（Export）は 設定 ➜ Export Hub。CSV 3 種・PDF 2 種から選び、生成 ➜ iOS 共有シート。PDF/CSV は Pro 機能。「・・・」➜ PDF タップ時は Pro ロックモーダル（v1.5）。v1.8 で Hub のキッカー（社内 PRD ID） / カード下の行数・サイズ / Share Sheet のサイズ / PDF プレビューの A4 表記 / フッターのバージョン番号 / Progress サブテキスト の 8 箇所の技術メタを一括除去。
⑨課金は 設定 ➜ プラン、または PRO 限定機能タップ時に Paywall。Pro 主軸は 外部活用（CSV/PDF/年次画像）＋ 体験（広告なし・テーマ）。v1.9 で「全データバックアップ Free/◎」を比較表に追加（GDPR Art.20 整合）。
⑩BackupScreen（v1.9 独立画面）：設定 ➜ バックアップ ➜ ZIP 書き出し / ZIP 取り込み（両方 Free・データ所有権の保証）。アンインストール前の保全・端末移行・他 OS への移行に使う。
⑪完全削除（v1.5）：設定 ➜ アーカイブ一覧 ➜ 項目を 長押し ➜ 確認モーダル「履歴も写真もすべて失われ、復元できません」➜ 完全削除。アーカイブだけでは消さない 2 段階ガード。
